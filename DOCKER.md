## Building Docker image

```bash
# Clone the repository
git clone https://github.com/RedSiege/C2Concealer.git
cd C2Concealer

# Build the Docker image
docker build -t C2concealer .

```

## Running with Docker

```bash
docker container run -it -v <cobalt_strike_location>:/usr/share/cobaltstrike/ c2concealer --hostname google.com --variant 3
```

## Copying generated profile from Docker
Make note of the profile name generated by C2concealer. In the example below, the profile is named d4402601.profile.

```
############################################################
# Profile successfully passed c2lintcheck                  #
# Profile Name: d4402601.profile                           #
# Generated by Red Siege's C2concealer tool.               #
############################################################
```

Retrieve the docker container ID using `docker ps -a`
```
CONTAINER ID   IMAGE                          COMMAND                  CREATED          STATUS                       PORTS     NAMES
1cb643c81dec   c2concealer                    "/C2concealer/.venv/…"   16 seconds ago   Exited (0) 11 seconds ago              reverent_taussig
```

With a profile name of `d4402601.profile` and a container ID of `1cb643c81dec`, to copy the generated profile to your current working directory, you would execute the following command:

```shell
docker cp 1cb643c81dec:/C2concealer/d4402601.profile .
```
